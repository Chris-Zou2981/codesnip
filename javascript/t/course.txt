# 我们的任务是让上面的海龟进行画图
# 先来看下最终的效果
# 请点击下面的运行按钮
speed 30
def box(length)
  for i = 1 to 4  
    fd length
    rt 90    
  end
end
for i = 1 to 18
  if i % 2 == 0 then
    pen "blue"
  else
    pen "red"
  end
  box 80
  lt 20
end

# 上一页的代码很多，我们先从最简单指令开始，让海龟前进一段距离，代码如下

fd 100

# 恭喜你，你可以让海龟前进了！
# 我们来学更多的命令，让海龟前进 100 步后，左转，再向前 100 步，然后右转，再向前 100 步

# lt 是 left turn 左转的意思，rt 是 right turn 右转的意思，后面的 90 时候转向的度数，90 度就是转个直角。

# 赞，你应该已经学会让海龟画三角形了！看起来编程也很简单，学会几个简单的指令就可以做一些事情了。
# 前面我们画的三角形边长是 50，如果我们想画一个边长为 60 的三角形，是不是要修改三个地方才能完成？
# 下面我们学习变量，让画出不同的三角形，只需修改一个地方即可。

lt 120

lt 120

# 上面的 steps = 50 就是定义了一个变量 steps 表示步数，后面所有 fd 指令的步数都使用了这个变量。
# 变量是个好东西，可以让我们的代码变更起来更简单，尝试下修改上面的代码，画一个边长为 100 的三角形吧。

# 我们已经会画三角形了，想不想来更多的三角形，现在我们来学习循环。
# 循环要用到 for 语句，就是让一些事情重复做 N 次，看下面的例子

steps = 100
for i = 1 to 3
  fd steps
  lt 120
  fd steps
  lt 120
  fd steps
end

# 可以看到上面的代码跟上一页的代码多了两行，for i = 1 to 3 表示我们要循环 3 次，而 end 表示我们循环执行的内容是 for 到 end 之间的代码。
# 细心的同学可以看到 for 循环里的代码都不是顶格写了，而是缩进了两个空格，这是一个好习惯，可以让代码看起来更清晰。
# 这次的小练习是复习变量，尝试修改上面的代码，把海龟每次旋转的度数提取成变量 d，改完后再运行代码，检查下是否正常。

# 给你点个赞，你能学到这里，应该是真的对编程很感兴趣了。
# 前面我们画的图形都是一个颜色，是时候让这个世界丰富多彩一些了。
# 改变画笔颜色需要一个新的指令 pen，我们需要给它指定一个颜色，颜色要用双引号括起来，如 pen "yellow"
# 再就是本节要学习一个新的 if 条件语句，它用来对某个条件进行判断，如果为真时，则执行指定的指令，如下。

steps = 100
d = 120
for i = 1 to 3
  if i == 2 then
    pen "blue"
  end
  fd steps
  lt d
  fd steps
  lt d
  fd steps
end

# 可以看到在 for 循环里，我们增加了一个 if 代码块，也是以 end 结尾的。
# if 和 then 直接有一个 i == 2 这里表示当前的 i 变量是否等于 2，如果等于 2 则把画笔变成蓝色。
# 运行结果看下，是不是有两个三角形变成蓝色了。
